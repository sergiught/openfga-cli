List objects of a type a user has a relation with

## Usage

```sh
ofga query list-objects [type] [relation] [user] [flags]
```

## Examples

```sh
  ofga query list-objects --type document --relation viewer --user user:anne
  ofga query list-objects document viewer user:anne
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `--context` | JSON object of condition context |   |
| `--contextual-tuple` | contextual tuple as user,relation,object (repeatable) | `[]` |
| `--relation` | relation |   |
| `--type` | object type |   |
| `--user` | user |   |
