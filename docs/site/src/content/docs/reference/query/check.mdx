Check whether a user has a relation on an object

## Usage

```sh
ofga query check [user] [relation] [object] [flags]
```

## Examples

```sh
  ofga query check --user user:anne --relation viewer --object document:roadmap
  ofga query check user:anne viewer document:roadmap
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `--context` | JSON object of condition context |   |
| `--contextual-tuple` | contextual tuple as user,relation,object (repeatable) | `[]` |
| `--object` | object (alternative to the positional arg) |   |
| `--relation` | relation (alternative to the positional arg) |   |
| `--user` | user (alternative to the positional arg) |   |
