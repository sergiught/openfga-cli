Run several checks in one request

## Usage

```sh
ofga query batch-check --check user,relation,object [...] [flags]
```

## Examples

```sh
  ofga query batch-check --check user:anne,viewer,doc:1 --check user:bob,editor,doc:1
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `--check` | a check as user,relation,object (repeatable) | `[]` |
