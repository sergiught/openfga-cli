Show tuple changes. By default all currently available changes are returned (the CLI auto-pages); --max-results (alias --limit) caps the total returned. --page-size only tunes the per-request page.

## Usage

```sh
ofga tuples changes [flags]
```

## Examples

```sh
  ofga tuples changes
  ofga tuples changes --type document
  ofga tuples changes --max-results 100
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `--limit` | alias for --max-results | `0` |
| `--max-results` | cap the total number of changes returned (0 = unbounded) | `0` |
| `--page-size` | per-request page size (0 = server default; not a total cap) | `50` |
| `--start-time` | only changes at/after this RFC3339 time |   |
| `--type` | filter changes by object type |   |
