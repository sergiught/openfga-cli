Write one relationship tuple, or many with --file

## Usage

```sh
ofga tuples write [user] [relation] [object] [flags]
```

## Examples

```sh
  ofga tuples write user:anne viewer document:roadmap
  ofga tuples write --user user:anne --relation viewer --object document:roadmap
  ofga tuples write --file tuples.json
  cat tuples.json | ofga tuples write --file -
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-n, --dry-run` | show the tuple that would be written without writing it | `false` |
| `--file` | JSON file of tuples to write in bulk ('-' for stdin) |   |
| `--object` | object (alternative to the positional arg) |   |
| `--relation` | relation (alternative to the positional arg) |   |
| `--user` | user (alternative to the positional arg) |   |
