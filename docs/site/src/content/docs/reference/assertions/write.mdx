Replace a model's assertions (the active --model-id, or the latest). The file is a JSON array of assertions or an object \{"assertions": [...]}, each: \{"tuple_key":\{"user","relation","object"},"expectation":true}.

## Usage

```sh
ofga assertions write --file <assertions.json> [flags]
```

## Examples

```sh
  ofga assertions write --file assertions.json
  ofga assertions write --model-id 01H… --file assertions.json
  cat assertions.json | ofga assertions write --file -
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-n, --dry-run` | validate the file and show what would be written without writing it | `false` |
| `-f, --file` | assertions JSON file ('-' for stdin) |   |
| `--force` | replace assertions without prompting | `false` |
