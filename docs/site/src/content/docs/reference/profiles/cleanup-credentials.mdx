Retry pending OS-keyring cleanup (or --purge all ofga secrets)

## Usage

```sh
ofga profiles cleanup-credentials [flags]
```

## Examples

```sh
  ofga profiles cleanup-credentials
  ofga profiles cleanup-credentials --purge
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-f, --force` | skip the confirmation prompt (with --purge) | `false` |
| `--purge` | delete ALL ofga secrets from the OS keyring, including orphans from deleted configs | `false` |
