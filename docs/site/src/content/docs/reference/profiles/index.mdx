Manage named connection profiles. Each profile stores an API URL, optional store and authorization-model IDs, and optional authentication settings.

| Command | Description |
| --- | --- |
| [add](add/) | Create a new profile |
| [cleanup-credentials](cleanup-credentials/) | Retry pending OS-keyring cleanup (or --purge all ofga secrets) |
| [current](current/) | Show the active profile name |
| [list](list/) | List all profiles |
| [remove](remove/) | Delete a profile |
| [set](set/) | Set a field on a profile |
| [show](show/) | Show a profile's resolved values (token masked) |
| [unset](unset/) | Clear a field on a profile |
| [use](use/) | Switch the active profile |
