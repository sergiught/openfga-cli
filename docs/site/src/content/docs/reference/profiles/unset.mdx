Clear a field on the active profile (or --profile).

`auth` (alias `auth_method`) clears the ENTIRE auth config — method, client_id,
token_url, audience and any keyring-stored secrets — not just the method. It
prompts for confirmation unless --force is given.

## Usage

```sh
ofga profiles unset <key> [flags]
```

## Examples

```sh
  ofga profiles unset store_id
  ofga profiles unset auth --force
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-f, --force` | skip the confirmation prompt when clearing auth | `false` |
