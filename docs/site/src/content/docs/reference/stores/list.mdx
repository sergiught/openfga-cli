List stores. By default all stores are returned (the CLI auto-pages); --max-results caps the total returned and stops paging once reached.

## Usage

```sh
ofga stores list [flags]
```

## Examples

```sh
  ofga stores list
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `--limit` | alias for --max-results | `0` |
| `--max-results` | cap the total number of stores returned (0 = unbounded) | `0` |
