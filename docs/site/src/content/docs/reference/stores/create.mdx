Create a new store

## Usage

```sh
ofga stores create <name> [flags]
```

## Examples

```sh
  ofga stores create my-store --use
```

## Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-n, --dry-run` | show what would be created without creating it | `false` |
| `--use` | save the new store ID to the active profile | `false` |
