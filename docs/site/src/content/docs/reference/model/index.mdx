Write, inspect and visualize authorization models

| Command | Description |
| --- | --- |
| [get](get/) | Show an authorization model as DSL |
| [graph](graph/) | Render the model as a colored relation graph |
| [latest](latest/) | Show the most recent authorization model as DSL |
| [list](list/) | List authorization models in the store |
| [test](test/) | Run authorization model tests against an embedded (or a real/versioned) OpenFGA server |
| [write](write/) | Write a new authorization model from a JSON or DSL (.fga) file |
