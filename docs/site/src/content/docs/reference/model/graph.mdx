Render an authorization model as a colored tree showing, for each type and relation, the directly-assignable types, implied relations, and inherited (tuple-to-userset) paths. With no argument, the latest model is used.

## Usage

```sh
ofga model graph [model-id]
```

## Examples

```sh
  ofga model graph
```
