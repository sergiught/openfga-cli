| [config](config/) | Inspect ofga's configuration |
| [init](init/) | Set up a connection profile (guided) |
| [model](model/) | Write, inspect and visualize authorization models |
| [playground](playground/) | Launch the interactive playground |
| [profiles](profiles/) | Manage connection profiles |
| [query](query/) | Ask authorization questions |
| [stores](stores/) | Create, list, inspect and delete stores |

| [tuples](tuples/) | Write, delete and read relationship tuples |
| [version](version/) | Print version and build information |
