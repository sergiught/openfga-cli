# homebrew-tap

Homebrew tap for [`ofga`](https://github.com/sergiught/openfga-cli) — a modern
CLI & TUI for OpenFGA.

> Copy this file into a new public repo named `homebrew-tap` under your account.
> The `Formula/ofga.rb` file is generated and pushed automatically by
> [goreleaser](https://goreleaser.com) on each `openfga-cli` release — you don't
> maintain it by hand.

## Install

```bash
brew install sergiught/tap/ofga
```

or

```bash
brew tap sergiught/tap
brew install ofga
```

## Upgrade

```bash
brew upgrade ofga
```
